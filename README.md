# NeoCoder Releases

Binary-only public distribution repository for NeoCoder.

The NeoCoder source repository is private. This repository is used only for GitHub Releases assets such as installer scripts, platform bundles, and checksums. It should not contain Rust source code, Cargo manifests, private CI logs, internal URLs, or credentials.

Installers and self-update default to this repository:

```bash
curl -fsSL https://github.com/H4nsa/neocoder-releases/releases/download/neocoder-v0.1.0/install.sh | sh
```
